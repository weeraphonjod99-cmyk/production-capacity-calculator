# Production Capacity Calculator

เว็บแอปสำหรับคำนวณกำลังการผลิตรายเครื่องจักร และสรุปกำลังการผลิตรวมของไลน์ผลิต

## ความสามารถ

- กรอกข้อมูลเครื่องจักรได้หลายรายการ
- คำนวณกำลังผลิตเป็นชิ้นต่อชั่วโมงและชิ้นต่อวัน
- สรุปกำลังผลิตรวม เครื่องคอขวด และ OEE เฉลี่ย
- แสดงกราฟเปรียบเทียบกำลังผลิตรายเครื่อง
- ติดตั้งเป็นแอปแบบ PWA ได้จาก browser
- ใช้งานออฟไลน์ได้หลังเปิดเว็บครั้งแรก
- บันทึกข้อมูลในเครื่องผู้ใช้ด้วย `localStorage`
- ส่งออกข้อมูลเป็นไฟล์ CSV

## สูตรคำนวณ

```text
กำลังผลิตต่อวัน =
((ชั่วโมงทำงานต่อวัน x 60 - เวลาหยุดต่อวัน) x 60 / Cycle Time)
x จำนวนชิ้นต่อรอบ x (OEE / 100) x จำนวนเครื่อง
```

```text
กำลังผลิตต่อชั่วโมง =
(3600 / Cycle Time) x จำนวนชิ้นต่อรอบ x (OEE / 100) x จำนวนเครื่อง
```

## วิธีเปิดใช้งาน

เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์ได้ทันที ไม่ต้องติดตั้ง dependency

## การติดตั้งเป็นแอป

หลังเปิดผ่าน GitHub Pages หรือเว็บเซิร์ฟเวอร์:

1. กดปุ่ม `ติดตั้งแอป` ในหน้าเว็บ ถ้า browser รองรับ
2. บนมือถือสามารถใช้เมนู browser แล้วเลือก `Add to Home Screen`
3. หลังติดตั้งแล้วเปิดใช้งานเหมือนแอปทั่วไป และใช้งานออฟไลน์ได้

## การนำขึ้น GitHub Pages

1. สร้าง repository ใหม่ใน GitHub
2. อัปโหลดไฟล์ `index.html`, `styles.css`, `app.js`, และ `README.md`
3. ไปที่ `Settings` > `Pages`
4. เลือก branch ที่เก็บไฟล์ เช่น `main`
5. เปิด URL ที่ GitHub Pages สร้างให้

## ไฟล์ในโปรเจกต์

- `index.html` โครงหน้าเว็บ
- `styles.css` หน้าตาและ responsive layout
- `app.js` สูตรคำนวณ การสรุปผล บันทึกข้อมูล และส่งออก CSV
- `manifest.webmanifest` ข้อมูลสำหรับติดตั้งเป็นแอป
- `service-worker.js` ระบบ cache สำหรับใช้งานออฟไลน์
- `icons/` ไอคอนแอป
- `README.md` คู่มือโปรเจกต์
